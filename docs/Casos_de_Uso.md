# Casos de uso – Gym Tracker Pro

## Actores del sistema
- Usuario registrado
- Administrador

---

## CASOS DE USO – USUARIO REGISTRADO

## CU-01 Registro de usuario
**Descripción:**  
El usuario se registra en la aplicación introduciendo sus datos personales.

**Actor:**  
Usuario registrado

---

## CU-02 Inicio de sesión
**Descripción:**  
El usuario accede a la aplicación mediante correo electrónico y contraseña.

**Actor:**  
Usuario registrado

---

## CU-03 Crear entrenamiento
**Descripción:**  
El usuario crea una nueva sesión de entrenamiento indicando la fecha y hora de inicio.

**Actor:**  
Usuario registrado

---

## CU-04 Añadir ejercicios a entrenamiento
**Descripción:**  
El usuario añade ejercicios a un entrenamiento, seleccionándolos del catálogo disponible.

**Actor:**  
Usuario registrado

---

## CU-05 Registrar series
**Descripción:**  
El usuario registra las series realizadas para cada ejercicio, indicando número de serie, repeticiones y peso.

**Actor:**  
Usuario registrado

---

## CU-06 Finalizar entrenamiento
**Descripción:**  
El usuario finaliza el entrenamiento indicando la hora de finalización y, opcionalmente, una valoración.

**Actor:**  
Usuario registrado

---

## CU-07 Consultar historial de entrenamientos
**Descripción:**  
El usuario consulta el listado de entrenamientos realizados y puede ver su detalle.

**Actor:**  
Usuario registrado

---

## CU-08 Consultar estadísticas personales
**Descripción:**  
El usuario visualiza estadísticas de volumen de entrenamiento y uso de grupos musculares.

**Actor:**  
Usuario registrado

---

## CASOS DE USO – ADMINISTRADOR

## CU-09 Gestionar músculos
**Descripción:**  
El administrador crea, edita o desactiva grupos musculares.

**Actor:**  
Administrador

---

## CU-10 Gestionar ejercicios
**Descripción:**  
El administrador crea, edita, activa o desactiva ejercicios, asignándolos a un grupo muscular y definiendo su dificultad.

**Actor:**  
Administrador

---

## CU-11 Consultar estadísticas globales (opcional)
**Descripción:**  
El administrador consulta estadísticas globales del sistema, como el número de usuarios registrados o los ejercicios más utilizados.

**Actor:**  
Administrador

---

## CU-12 Gestionar usuarios
**Descripción:**  
El administrador consulta el listado de usuarios registrados y puede activar o desactivar sus cuentas, controlando el acceso al sistema sin eliminar la información asociada.

**Actor:**  
Administrador
