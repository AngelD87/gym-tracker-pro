# Casos de uso - Gym Tracker PRO

## Actor principal
Usuario registrado 

---

## CU-01 Registro de Usuario
El usuario se registra en la apliacación introduciendo sus datos personales.

**Actor:**
Usuario registrado

----

## CU-02 Inicio de sesión
**Descripción:**
El usuario accede a la apliación mediante correo electrónico y contraseña

**Actor:**
Usuario registrado

---

## CU-03 Registrar entrenamiento
**Descripción:**
El usuario crea una nueva sesión de entrenamiento indicando la fecha

**Actor:**
Usuario registrado

---

## CU-04 Añadir ejercicios y series
**Descripción:**
El usuario añade ejercicios a un entrenamiento y registra las series realizadas.

**Actor:**
Usuario registrado

---

## CU-05 Consultar historial
El usuario consulta el historial de entrenamientos realizados.

**Actor:**
Usuario registrado

---

## CU-06 Consultar estadísticas
**Descripción:** 
El usuario visualiza estadísticas básicas de su entrenamiento y volumen muscular.

**Actor:**
Usuario registrado
